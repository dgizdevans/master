# Master Repository

This repository contains labs, assessments, and thesis research from my
master's programme in **Data Analytics and Artificial Intelligence**
(Double Degree: TSI Riga × UWE Bristol).

  Master's thesis research: *Heterogeneous Graph Neural Networks for User

- **[AI Project](https://github.com/dgizdevans/master/tree/main/ai_project)** —

- **[Big Data](https://github.com/dgizdevans/master/tree/main/big-data)** —

- **[Data Mining](https://github.com/dgizdevans/master/tree/main/data-mining)** —
  Lab exercises on data analytics using Python

- **[Machine Learning](https://github.com/dgizdevans/master/tree/main/ml)** —
  Lab exercises focused on machine learning

- **[ML Coursework](https://github.com/dgizdevans/master/tree/main/ml_coursework)** —
